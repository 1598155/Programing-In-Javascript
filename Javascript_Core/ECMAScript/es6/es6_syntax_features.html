ES6语法特性

ES6包含了很多万众期待的特性支持：

  • arrow functions
  • const
  • let
  • default function params
  • rest parameters
  • call(…)
  • array(…)
  • class
  • computed properties
  • modules
  • for…of
  • Array comprehensions
  • Generator comprehensions
  • Iterators
  • yield
  • Template Strings
  • block-level declaration
  • destructing
  • promoise
