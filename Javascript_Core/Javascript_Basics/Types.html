Javascript数据类型

Javascript没有强制对象类型检测，但不意味着没有数据类型。有两类：

  • 原始类型
  • 对象类型

这两类对象在使用、参数传递、返回值上都有细微差别。

字面量

  • NullLiteral
  • BooleanLiteral
  • NumbericLiteral
  • StringLiteral
  • RegularExpressionLiteral
  • ObjectLiteral

原始数据类型

  • undefined
  • null
  • Boolean value
  • Number value
    • NaN
    • Inifinity
  • String value

对象数据类型

  • Number object
  • String object
  • Boolean Object
  • Function Object
  • RegExp Object
  • Error
    • SyntaxError
    • ReferenceError
    • TypeError
  • …

隐式转换

  • null == undefined
  • undefined == null
  • 2 == “2”
  • “2” == 2
  • NaN != NaN
